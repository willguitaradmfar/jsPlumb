Projeto WLPFLOW
==========
Projeto para construção de aplicações nodejs padronizadas

Como usar ?
===========

```bash
npm install
```

```bash
npm start
```

==============

* [http://localhost:8081/scanBox](http://localhost:8081/scanBox) Para Scanear os boxes
